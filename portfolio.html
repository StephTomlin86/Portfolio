Day Planner

Using jQuery, CSS and HTML, I created a simple calendar application that allows the user to save events for each hour of the work day. This is a web-based application and required using Moment.js libary to work with the date and time. The grey color blocks indicate hours that have already passed, red indicates the current hour, and green expresses future time blocks. This was my first experiences working with and manipulating local storage.

Weather or Not: A Park Finder

In this group project, my team and I wanted to craft a web based application that was meaningful, and could potentially improve the quality of living for many who are experiencing the trying times of quarantine during the COVID-19 pandemic. While both individuals and families have been experiencing isolation and regular activities are no longer available, the beauty of nature within our state park system has remained accessible. By developing a state park finder that can assist users in accessing safe and enriching activities, Weather or Not also provides an accurate weather forecast to ensure a quality outdoor experience, so you always know whether or not you should plan for a different day to visit the parks.

A Code Quiz Challenge

In this assignment, the goal was to create a timed, multiple choice quiz. When the user answers a question incorrectly, the timer automatically deducts 10 seconds from the timer. When the user answers a question correctly, then the points accumulate by 10. When the game is over, then the player can save their initals and score and then save that information in the browser.
